# Raven OS

>Control your system, not the other way around.

Raven OS is a privacy-first, performance-focused operating system built for the Project Crow ecosystem.

It is designed to feel fast, clean, and intentional — combining the efficiency of modern ARM hardware with a user experience that puts control back in the hands of the user.

---

Vision

Modern operating systems are bloated, restrictive, and built around companies — not users.

Raven OS exists to change that.

We are building:
- A clean, minimal OS with no unnecessary background processes
- A system that respects privacy by default
- An experience that is deeply integrated with its hardware

Raven OS is not just another Linux distro — it is the foundation for a new ecosystem.


Core Features (Planned)

 Flight Controller
A real-time system control panel that gives users direct visibility and control over:
- CPU (performance & efficiency cores)
- System performance modes
- Background process behaviour



Obsidian UI
- Dark-first design (black base, white text)
- System-wide accent colour control
- Clean, minimal interface
- No clutter, no unnecessary elements



Familiar, but Better
- Intuitive like Windows
- Smooth like macOS
- Flexible like Linux



Privacy First
- No telemetry
- No tracking
- Hardware-level privacy support (camera kill switch, etc.)



Hardware Integration

Raven OS is being designed alongside custom hardware:

### **Rook (by Project Crow)**
- ARM-based laptop
- Optimised for efficiency and performance
- Built specifically for Raven OS
