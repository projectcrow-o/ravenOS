## ЁЯРжтАНтмЫ What is Raven OS?

**Raven OS** is an independent operating system developed under **Project Crow** тАФ not a Linux distro, not a Windows reskin. Something built from the ground up with a clear set of values: performance, privacy, and putting control back in the user's hands.

It is designed to feel instantly familiar тАФ borrowing the best interaction patterns from Windows and macOS тАФ while running leaner, cleaner, and more honestly than either of them.

Raven OS is being developed alongside dedicated Project Crow hardware, so the software and the machine it runs on are built to understand each other from day one.

> **"Your hardware. Your data. Your OS."**

## ЁЯЦея╕П The Problem

Most people don't realise how much their OS costs them тАФ not in money, but in performance, battery life, and privacy.

| | Windows | macOS | Raven OS |
|---|---|---|---|
| Background bloat | Extreme | Moderate | None |
| Telemetry | Always on | Partial | Zero |
| Hardware freedom | Any (messy) | Apple only | Project Crow optimised |
| User control | Low | Very low | Full |
| Privacy by default | No | No | Yes |

Windows ships ads, telemetry, and background processes you never asked for. macOS is polished but locked to Apple's ecosystem and Apple's rules. Linux gives you freedom but asks you to earn it. Raven OS sits in the space none of them occupy.

## тЬж Core Features

### Flight Controller
A system-level process manager built natively into the OS. See every core in real time. Lock your primary application to performance cores. Push background tasks to efficiency cores. Switch between **Performance**, **Balanced**, and **Battery Saver** modes in one click тАФ no third-party tools, no digging through settings.

### Obsidian UI
Deep black base. White on dark by default. A system-wide accent colour engine that propagates a single colour choice through every UI element тАФ window borders, the taskbar, highlights, status indicators. Clean, intentional, consistent.

### Floating Taskbar
A centred, floating dock at the bottom of the screen that expands and contracts fluidly as applications open and close. No clutter. No tray icons you didn't ask for.

### Privacy by Default
No telemetry. No usage reporting. No background data collection. The Project Crow hardware Raven OS is designed for includes a **hardware-level camera kill switch** тАФ not a software block, an actual power cut to the camera module. Privacy is a design decision, not a setting.

### Familiar, Not Foreign
A top bar for system status. A taskbar. Right-click menus that behave the way you expect. A file manager that makes sense. You should not need to relearn how to use a computer тАФ Raven OS meets users where they are and gets out of the way.

### MDM-Ready Architecture
Ships clean for personal users. For organisations тАФ schools, businesses, institutions тАФ Raven OS exposes a device management API compatible with standard MDM tooling. No surveillance layer baked in, just clean hooks for IT teams that need them.
